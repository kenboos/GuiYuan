## 1. 项目概述

### 1.1 产品定位
归园是一款连接线上虚拟农场与线下真实地块的微信小程序，用户可在线选地、充值C币、装饰地块、选择种植作物及托管服务，实现"指尖种地，线下收获"的田园体验。

### 1.2 技术栈
- **框架**：uni-app（Vue3 + TypeScript）
- **UI组件**：uni-ui + 自定义组件
- **地图**：腾讯地图 SDK
- **支付**：微信支付
- **状态管理**：Pinia
- **HTTP客户端**：uni.request 封装

## 2. 核心功能模块

### 2.1 用户登录模块

#### 功能描述
微信一键登录，获取用户openid，建立用户体系。

#### 页面路径
`pages/index/index`

#### 核心交互
1. 进入小程序 → 调用 `wx.login()` 获取 code
2. 发送 code 到服务端 `/api/auth/login`
3. 服务端返回 token + 用户信息
4. 本地存储 token，跳转首页

#### API接口
```typescript
// 登录
POST /api/auth/login
Request: { code: string }
Response: { token: string, user: UserInfo }

// 获取用户信息
GET /api/user/info
Headers: Authorization: Bearer {token}
Response: UserInfo
```

#### 数据结构
```typescript
interface UserInfo {
  id: string;
  openid: string;
  nickname: string;
  avatar: string;
  cCoin: number;           // 当前C币余额
  plotCount: number;       // 拥有地块数
  createdAt: string;
}
```

### 2.2 地块选择模块

#### 功能描述
可视化地图展示所有地块，支持筛选、查看详情、购买。

#### 页面路径
`pages/plots/plots` - 地块列表/地图  
`pages/plot-detail/plot-detail` - 地块详情

#### 核心交互
1. 进入页面 → 加载地块地图数据
2. 支持切换「地图模式」/「列表模式」
3. 点击地块 → 查看详情（面积、价格、状态）
4. 可购买地块 → 跳转支付确认

#### 页面元素
- 顶部：搜索栏、筛选器（面积/价格/状态）
- 中部：腾讯地图组件 / 地块卡片列表
- 底部：地块状态图例

#### API接口
```typescript
// 获取地块列表
GET /api/plots
Query: { status?: 'available' | 'sold', minArea?: number, maxArea?: number }
Response: Plot[]

// 获取地块详情
GET /api/plots/:id
Response: PlotDetail

// 购买地块
POST /api/plots/:id/purchase
Request: { decorationId?: string }
Response: { orderId: string, paymentParams: PaymentParams }
```

#### 数据结构
```typescript
interface Plot {
  id: string;
  name: string;           // 地块编号，如"A-01"
  area: number;           // 面积（平方米）
  price: number;          // 年租价格（C币）
  status: 'available' | 'sold' | 'maintenance';
  location: {
    lat: number;
    lng: number;
  };
  image: string;          // 实景图
}

interface PlotDetail extends Plot {
  description: string;
  soilType: string;       // 土壤类型
  sunlight: string;       // 光照条件
  surrounding: string[];  // 周边设施
  currentOwner?: UserInfo; // 当前拥有者（如有）
}
```

### 2.3 C币充值模块

#### 功能描述
C币是平台虚拟货币，1元=10C币，支持多档充值，梯度赠送。

#### 页面路径
`pages/recharge/recharge`

#### 充值档位
| 实付金额 | 到账C币 | 赠送比例 |
|---------|--------|---------|
| ¥10 | 100 C | — |
| ¥30 | 320 C | +6.7% |
| ¥100 | 1100 C | +10% |
| ¥300 | 3600 C | +20% |
| ¥1000 | 13000 C | +30% |

#### 核心交互
1. 选择充值档位 → 显示到账C币
2. 点击支付 → 调起微信支付
3. 支付成功 → C币实时到账

#### API接口
```typescript
// 创建充值订单
POST /api/coin/recharge
Request: { amount: number, tier: string }
Response: { orderId: string, paymentParams: PaymentParams }

// 查询充值记录
GET /api/coin/transactions
Query: { page: number, limit: number }
Response: Transaction[]

// 查询C币余额
GET /api/coin/balance
Response: { balance: number }
```

#### 数据结构
```typescript
interface Transaction {
  id: string;
  type: 'recharge' | 'purchase' | 'refund';
  amount: number;         // 变动金额（正为增加，负为减少）
  balance: number;        // 变动后余额
  description: string;    // 描述
  createdAt: string;
}
```


#### 数据结构
```typescript
interface DecorationItem {
  id: string;
  name: string;
  type: 'fence' | 'landscape';
  material: string;
  price: number;
  image: string;
  previewImage: string;   // 预览图
}

  itemId: string;
```

### 2.6 托管服务模块

#### 托管套餐
| 套餐 | 年费（C币） | 服务内容 |
|-----|-----------|---------|
| 基础托管 | 800 | 浇水、除草、基础施肥 |
| 标准托管 | 1500 | 基础+病虫害防治、定期修剪 |
| 尊享托管 | 2800 | 标准+专属管家、24h监控、优先配送 |

#### 核心交互
1. 选择要托管的地块
2. 查看套餐对比 → 选择套餐
3. 确认支付年费 → 托管生效
4. 可随时查看托管服务记录

#### API接口
```typescript
// 获取托管套餐列表
GET /api/hosting/packages
Response: HostingPackage[]

// 获取地块托管状态
GET /api/plots/:id/hosting
Response: HostingStatus

// 购买托管服务
POST /api/plots/:id/hosting
Request: { packageId: string, year: number }
Response: { orderId: string, totalCost: number }

// 获取托管服务记录
GET /api/plots/:id/hosting/logs
Query: { page: number, limit: number }
Response: HostingLog[]
```

#### 数据结构
```typescript
interface HostingPackage {
  id: string;
  name: string;
  pricePerYear: number;   // 年费（C币）
  services: string[];     // 服务内容列表
  description: string;
}

interface HostingStatus {
  plotId: string;
  package: HostingPackage;
  startDate: string;
  endDate: string;
  isActive: boolean;
}

interface HostingLog {
  id: string;
  date: string;
  service: string;
  description: string;
  images: string[];       // 服务现场照片
  staffName: string;      // 执行人员
}
```

## 4. 接口联调规范

### 4.1 请求封装
```typescript
// utils/request.ts
const request = (options: RequestOptions) => {
  return new Promise((resolve, reject) => {
    uni.request({
      ...options,
      header: {
        'Authorization': `Bearer ${getToken()}`,
        'Content-Type': 'application/json'
      },
      success: (res) => {
        if (res.statusCode === 200) {
          resolve(res.data);
        } else if (res.statusCode === 401) {
          // Token过期，重新登录
          reLogin();
        } else {
          reject(res.data);
        }
      },
      fail: reject
    });
  });
};
```

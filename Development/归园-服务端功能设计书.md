## 1. 项目概述

### 1.1 技术栈
- **框架**: Node.js + Fastify
- **语言**: TypeScript
- **数据库**: MySQL 8.0 + Redis
- **ORM**: Prisma
- **认证**: JWT
- **支付**: 微信支付
- **文件存储**: 腾讯云 COS

### 1.2 目录结构
```
src/
├── config/          # 配置
├── controllers/     # 控制器
├── services/        # 业务逻辑
├── models/          # Prisma schema
├── middlewares/     # 中间件
├── routes/          # 路由
├── utils/           # 工具
├── jobs/            # 定时任务
└── admin/           # 管理后台
```

## 4. 核心业务逻辑

### 4.1 登录流程
```
小程序 wx.login()获取code
    ↓
POST /api/auth/login { code }
    ↓
服务端用code换openid（调微信接口）
    ↓
查询/创建用户记录
    ↓
生成JWT token返回
```

### 4.2 地块购买流程
```
用户选择地块
    ↓
POST /api/plots/:id/purchase
    ↓
检查地块状态（必须available）
    ↓
创建订单（类型：plot_purchase）
    ↓
返回支付参数
    ↓
调起微信支付
    ↓
支付成功回调 → 更新订单状态 → 更新地块owner → 扣减C币（如用C币支付）
```

### 4.3 C币充值流程
```
用户选择充值档位
    ↓
POST /api/coin/recharge { amount, tier }
    ↓
创建充值订单（类型：coin_recharge）
    ↓
调起微信支付（人民币）
    ↓
支付回调 → 增加用户C币余额 → 创建transaction记录
```

### 4.4 装饰购买流程
```
用户编辑装饰方案
    ↓
POST /api/plots/:id/decoration { items }
    ↓
计算总价 → 检查C币余额
    ↓
创建装饰订单 → 扣减C币
    ↓
生成线下执行单（管理员可见）
```

### 4.5 托管服务流程
```
用户选择托管套餐
    ↓
POST /api/plots/:id/hosting { packageId, year }
    ↓
计算年费总价 → 检查C币余额
    ↓
创建托管订单 → 扣减C币
    ↓
创建hosting记录（设置起止时间）
    ↓
托管到期前30天推送续费提醒
```
